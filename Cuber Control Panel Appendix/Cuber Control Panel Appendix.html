Cuber Control Panel Home Screen

From the home screen, select “Screen Selection” in the bottom right corner of the screen to have the menu options appear. Selecting “Machine Setup” brings up the machine setup screen where you can manually change the patternmaker, increase the speed, or find tune a pattern.

From the home screen, selecting the “Cube Codes” will bring up the list of codes/options from which you can choose from.

Selecting “Screen Selector” will bring a list of options at the bottom of the screen.

Selecting “Machine Setup” will bring you to a screen with Busser Company phone number as well as important machine information and codes.

Selecting “Program tier” allows you to create a single tier without it being apart of a cube.

Selecting “Change Cube to Run” is where you change the cube pattern from one to another by selecting the number you would like.

Selecting “Program Cubes” allows you to program a whole entire cube.

Selecting Error Log” is where you can find a list of all the errors that have occurred including a description of the error, and the date and time.

Selecting “Product Report”

Selecting “IO Link Status” will allow the Busser company to remotely control the machine. This feature is for technical support.

Selecting “Data Access” allows you to diagnose a problem the machine is having.

Selecting “Master Reset” will allow you to completely reset the machine.

Selecting “IOS Status” allows you to view which parts of the machine are “on” or “off”.

Selecting “Manuel Mode” Allows you to manually control any part of the machine. First you must put both the “Cuber” and “Patternmaker” to manual.
